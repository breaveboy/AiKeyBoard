1. 基本电气参数
参数	数值
供电电压 VDD	典型 5V，范围 3V ~ 5.5V
单色恒流	典型 5mA，范围 4mA ~ 6mA
高电平输入 VIH	≥ 0.6 × VDD
低电平输入 VIL	≤ 0.3 × VDD
PWM 频率	典型 16kHz
静态功耗	典型 0.45mA
最大数据速率 FDIN	1000kHz
传输延迟 DIN → DOUT	最大 200ns

这些来自规格书的电气参数和开关特性表。

2. 引脚定义
引脚	名称	作用
VDD	电源	供电
DOUT	数据输出	级联到下一颗灯
DIN	数据输入	MCU 数据输入端
GND	地	电源地和信号地

规格书说明每个灯珠内部带有数据锁存、整形放大和级联转发功能。每颗灯会取走前面 24bit 数据，剩余数据从 DOUT 输出给下一颗灯。

3. 通信协议参数

这个灯使用的是 单线归零码协议，和 WS2812 类似。

参数	说明	典型值	容差
T0H	0 码高电平时间	0.295us	±0.05us
T0L	0 码低电平时间	0.595us	±0.05us
T1H	1 码高电平时间	0.595us	±0.05us
T1L	1 码低电平时间	0.295us	±0.05us
Trst	复位低电平时间	≥80us	—

也就是说：

0码：高 0.295us + 低 0.595us
1码：高 0.595us + 低 0.295us
Reset：低电平 ≥80us

规格书第 10 节“数据通信协议”明确写的是单极性归零码，每个码元起始为高电平，通过高电平宽度区分 0 和 1。

4. 数据格式

每颗灯需要 24bit 数据：

G7 G6 G5 G4 G3 G2 G1 G0
R7 R6 R5 R4 R3 R2 R1 R0
B7 B6 B5 B4 B3 B2 B1 B0

也就是顺序是：

GRB

不是 RGB。

例如点红色：

R = 255
G = 0
B = 0

实际发送顺序是：

G = 0x00
R = 0xFF
B = 0x00
